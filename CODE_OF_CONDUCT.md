# Code Of Conduct

DevMesh is committed to a welcoming, respectful, and practical collaboration environment for contributors and users.

## Our Commitment

- Welcome contributors from different backgrounds and experience levels.
- Keep technical discussion respectful, specific, and kind.
- Focus criticism on ideas, behavior, and implementation details, never on personal attributes.
- Protect private information, credentials, and sensitive project data.

## Expected Behavior

- Be collaborative in issues, pull requests, reviews, and discussions.
- Prefer evidence, reproducible examples, and clear reasoning.
- Help newcomers get oriented instead of gatekeeping.
- Respect repository contribution rules, review boundaries, and release processes.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Publishing credentials, private logs, or personal information without consent.
- Repeated bad-faith disruption, spam, or abusive review behavior.

## Reporting

For repository-related concerns, contact the current project lead through:

- GitHub: https://github.com/xy200303
- Repository issues: https://github.com/xy200303/DevMesh/issues

This file can evolve as the contributor community grows and governance needs become clearer.
